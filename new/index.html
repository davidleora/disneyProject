This is a project that I made to practice my programming skills.

It is a simple web application that allows you to search for Disney characters.

The data is fetched from the Disney API. The application also has a music player that plays Disney music.

You can choose from different themes and control the volume. I hope you enjoy it!


It is still under construction and has a lot of bugs, I am planning to update it when I have free time!
